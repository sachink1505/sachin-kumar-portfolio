## Deployment (GitHub Pages — FREE)

1. Go to github.com → Sign up → New repository → name it `yourusername.github.io`
2. Drag and drop ALL files (index.html + images folder) into the upload area
3. Click "Commit changes"
4. Go to Settings → Pages → Source: "Deploy from branch" → Branch: main → Save
5. Your site is live at `https://yourusername.github.io` in 1-2 minutes

## Making Changes

**Edit text:** Click index.html on GitHub → pencil icon → Ctrl+F to find text → edit → Commit
**Add a photo:** Upload to the `images/` folder, then reference it in the HTML
**Change colors:** Search for `--accent:#c9553d` and replace with any hex color
